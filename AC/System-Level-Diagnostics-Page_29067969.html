JIRA Issue(s) ARF-384 - AccMan 'Health Check' Page Closed

Details

Need to add a page to display various reports, statistics and health checks relating to Accman data and systems that Accman interfaces with.

This page should be accessible from the Accman home page, and to begin with should display a recent history of the last few Banner and SAP imports, and whether they were successful or not (exact number TBC).

It should also have the ability to show (perhaps on a separate tab) general information about transactions such as how many 'reset password' transactions are currently in a pending state. This can later be used to flag an early warning up if the number get's too high. It should be generic so that the report can be used for any type of transaction.
