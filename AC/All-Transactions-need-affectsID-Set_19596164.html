Details

A number of transactions do not have the affectsID set. Following SQL returns 9 transactions:

 

Select count(tr.tr_type_id),tr.tr_type_id,tt.tr_description
from transactions tr
join trtypes tt on tr.tr_type_id = tt.tr_type_id
where tr.affectsID = 0 or tr.affectsID is null
group by tr.tr_type_id;
