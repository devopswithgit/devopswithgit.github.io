How to Generate Definitions CSV Files from a DLL

To regenerate the above list use the following Power Shell 2.0 script

Generate_DLL_Reference_CSV.ps1

  1. Download the Generate_DLL_Reference_CSV.ps1 script above and save to C:\dev\ (the following assumes you download to C:\dev\)
  2. Run PowerShell 2.0 as Administrator
  3. To load and run the script type at the PS :\> command then press Enter:
    C:\dev\Generate_DLL_Reference_CSV.ps1
  4. At the prompt "Type the directory path to create CSV:" type
    C:\dev\DLL_Reference_Docs\
  5.  At the prompt "Please choose a name for the CSV file:" type a relevant name to the DLL being documented:
    e.g.
  6. At the prompt "Type the path to the DLL to document:" type the path where the DLL is:
    e.g. 
  7. Press Enter. You will see all the content is then written to the "AccManLibrary.csv" file (or your chosen one).
  8. Type:
    Exit
    and press Enter to finish.
