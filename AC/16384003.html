Accman : Accman Front-End - displaying student "level" (UG/PG)

Created by Unknown User (ais008), last modified on Sep 04, 2013

At present in the Accman front-end, when the search results list includes students, it can sometimes show postgrad students as "Undergraduate" and vice versa.  This is because it is deriving the student level from the accman account table rather than the person table.  Up until around July 2012, students used to be issued with a new username when they changed level or school in Banner (usually from UG to PG but potentially also the opposite direction); in July 2012 the account assignment logic handling the Banner SYRACCS table was changed so that a student with an existing UG username would keep that username if they became PG.

The logic in the Accman front end search results screen needs to be changed to use Mail.person.stu_levl (or Mail.person.person_status_code might be better) instead of using Mail.account.status_code.
