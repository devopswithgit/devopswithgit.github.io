Accman : ElasticSearch

Created by Unknown User (aaa45815), last modified on May 16, 2016

ElasticSearch is an open source indexing engine based on Apache's Lucene. It ahs a high degree of flexibility with regards to the indexing options. The logstash default configuration tends towards indexing the entire document (or event in logstash parlance) to allow simple full text searching to be performed with minimal configuration.

We have the option of writing a custom front end that provides a more natural workflow into investigating issues that would also hide the query language that is implemented by elastic search. Kibana is available out of the box to provide powerful search and visualisation tools too.
