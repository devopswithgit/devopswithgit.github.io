Accman : Accman Front-End Access Levels

Created by Paul Daniels, last modified on Dec 22, 2015

The Accman Front-End, url https://accmanweb.salford.ac.uk/AccManFE/Account/LogOn, provides a number of different access levels via the operators table in the Mail database.

AccessLevelProvides
1

Find Users

Options to Enable A/c, Disable A/c, Extend A/c, Change Password, Generate Password File, Unlock AD A/c,

Delivery Notification Method, Extend Borrower Status,

Change O365 Password, Disable O365 A/c, Enable O365 A/c

Associates

Can Bulk Upload, Create Cohorts, Create Associates, Create Classes

2

Find Users

Options to Enable A/c, Disable A/c, Extend A/c, Change Password, Generate Password File, Unlock AD A/c,

Delivery Notification Method, Extend Borrower Status,

Change O365 Password, Disable O365 A/c, Enable O365 A/c

Associates

Can Bulk Upload, Create Cohorts, Create Associates, Create Classes

3

Find Users

Delivery Notification Method, Extend Borrower Status

Associates

Can Bulk Upload, Create Cohorts, Create Associates, Create Classes

4

Find Users

None.

Associates

Cannot Create Associates, Classes or upload Cohorts.

5

Find Users

None.

Associates

Cannot Create Associates, Classes or upload Cohorts.

6

Find Users

None.

7

Associates

Cannot Create Associates, Classes or upload Cohorts.

8

Find Users

None.

Associates

Cannot Create Associates, Classes or upload Cohorts.

9

Find Users

None.

Associates

Cannot Create Associates, Classes or upload Cohorts.

 

It is not known if there are more levels then the first nine.
