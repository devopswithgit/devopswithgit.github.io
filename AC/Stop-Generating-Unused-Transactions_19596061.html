There are still some unused transactions being generated and not processed. Using the following SQL, I get 7 transaction types:

 

Select count(tr.tr_type_id),tr.tr_type_id,tt.tr_description
from transactions tr
join trtypes tt on tr.tr_type_id = tt.tr_type_id
where tr.tr_state = 'waiting'
group by tr.tr_type_id;
