Accman : Office365 Alumni License Not Running

Created by Stephen Harrison, last modified on Oct 12, 2015
SummaryOffice365 Alumni License Not Running 
Issue TypeBug
PriorityHigh
Raised ByStephen Harrison
Date Raised09/10/2015
SourceSteve Harrison 
JIRA Issue(s) ARF-215 Closed

 

Details

  • Historical background: The code was originally developed when Microsoft changed their licensing policy for Office365 so that 'current active student' should be given one type of Office365 license and that Alumni should have another type.  The code was deployed circa July 2013 but never enabled / executed.  Circa Nov 2014 this was switched on in response to an incident but subsequently caused havoc, in the main by providing Alumni licenses to active post graduate students, and therefore switched off again once this situation had been reversed.
  • The suggestion is that we should switch this back on now but only after all pending transactions have been removed and that this action has been tested in UAT to confirm that upon initial implementation no students are actually re-licensed but would only be re-licensed as they become an Alumni at a future date.
