Introduction

Amend AccMan so that when new user accounts are created the following AD attribute is updated:

AD Attribute => scriptPath

Value => UoS-Login.vbs”

 

required changes: 

ActiveDirectoryBaseProcessor:
in setStandardAttributes(): adAttrs.Add("scriptPath", "UoS-Login.vbs");

 

To test, ensure that the above attribute is written out for a newly created user.
