Accman : Office 365 Deployment Notes

Created by Julia Kelbrick, last modified on Aug 15, 2013

 

 

Deployment of Office 356 PowerShell Processors on our Servers

·         Download and install Microsoft Online Services Sign-In Assistant for IT Professionals BETA from:

http://www.microsoft.com/en-us/download/details.aspx?id=39267

 

·         Download and install Windows Azure Active Directory Module for Windows Powershell (64-bit version) from:

http://technet.microsoft.com/en-us/library/jj151815.aspx#BKMK_Requirements

 

·         Modify the C:\accman\lib\paths.pl Perl Script.  Change the $powershell_data_dir value to:

D:\accman\PowershellData\O365

·         Install the Powershell Scripts for Office 365 to these folders on the AccMan BackEnd Production Server):

C:\accman\PowershellScripts\Office365Processors

 

·        


Deployment of Office 356 Password Strengthening Modifications

The following Perl Script has been modified so that AccMan generates a 9 character password.  Deploy the modified versions of the scripts to the

AccMan BackEnd Production Server  (NB Check if there are any other Production AccMan servers it needs deploying to):

C:\accman\scripts\register.pl

Perl code changed from this . . . . .

 

#=====================================================================================

sub generateEmailPassword {

  # generate 6 character password from 32-character alphabet (excludes I, O, 0, 1)

  my @alphabet = qw{A B C D E F G H J K L M N P Q R S T U V W X Y Z 2 3 4 5 6 7 8 9};

 

  # generate password

  my $password = '';

  for (0 .. 5) {

    $password .= $alphabet [int rand 32];

  }

  return $password;

}

 

To this . . . . . .

 

#=====================================================================================

sub generateEmailPassword {

  # generate 9 character password from 58-character alphabet (excludes I, O, 0, 1)

  # Passwords for Office 365 should be 9-16 characters and require at least 3 of the following: lowercase, uppercase, numbers (0-9), defined set of symbols

 

  # generate password

  my $password = '';

 

  # First 3 characters are random from full character list

  my @alphabet = qw{a b c d e f g h i j k l m n o p q r s t u v w x y z A B C D E F G H J K L M N P Q R S T U V W X Y Z 2 3 4 5 6 7 8 9};

  for (0 .. 3) {

    $password .= $alphabet [int rand 58];

  }

 

  # The fourth character of password is uppercase

  my @alphabet = qw{A B C D E F G H J K L M N P Q R S T U V W X Y Z};

  $password .= $alphabet [int rand 24];

 

  # Fifth character of password is numeric

  my @alphabet = qw{2 3 4 5 6 7 8 9};

  $password .= $alphabet [int rand 8];

 

  # The next 3 characters are random from full character list

  my @alphabet = qw{a b c d e f g h i j k l m n o p q r s t u v w x y z A B C D E F G H J K L M N P Q R S T U V W X Y Z 2 3 4 5 6 7 8 9};

  for (0 .. 3) {

    $password .= $alphabet [int rand 58];

  }

 

  # The final character of password is lowercase

  my @alphabet = qw{a b c d e f g h i j k l m n o p q r s t u v w x y z};

  $password .= $alphabet [int rand 26];

 

  return $password;

}

 

Deploy the modified script to the AccMan BackEnd Production server(s).
