Accman : ISA006 - Access to Library on change of Interruption Status

Created by Jeremy Sugden, last modified on Jan 06, 2016

Supporting Context

When the student stops interrupting the existing process will provide the student ID card with the correct level of access for that student/registration status. The student ID card will continue to be disabled if the student does not return to study or will attract full student access if they do. Any change to the process must ensure that current functionality is maintained.

The Interrupters temporary card  accounts will be disabled manually by the library staff, using the same process as that used for external borrowers. A returning student will be informed of the disabled interrupters temporary account and the activation of their full student account by (a manually produced) email from Student Admin. This process is wholly out of scope for the ITS solution.

A report of all students whose student and/or registration status  changes from ‘IN’ between 2 (inclusive) dates is required to identify interrupters temporary cards that need to be disabled. For each student identified the following information is required:

  • Roll number
  • First name
  • Last name
  • University email address
  • Registration Status
  • Student Status
  • Last Registration status change date.  
  • Last Student status change date

This report will be needed on an ad hoc basis by the Library team to identify cards that need to be disabled. Reports will not be run in the period after roll-over and before registration as this will not provide reliable information.
