Accman : Determine Students' 'final year' status

Created by Unknown User (ujs077), last modified by Unknown User (uis894) on Nov 14, 2017

Introduction

Accman will set the Leaver flag of person table to either 'Y' or 'N'. This is based on following flowchart.

When ALMA check this flag with student level, Final year status comes out. For example if Student Leaver flag is set to Y with Student Level Set to UG, will get 'Final Year' Status.

See  ARF-374 - Review AccMan algorithm for determining students 'final year' status Closed  for an issue that the current logic appears to be causing.
