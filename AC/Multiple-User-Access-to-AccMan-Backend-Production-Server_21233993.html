Details

  • Currently, any members of the Development or Support teams who need to access the AccMan Backend Production server must take turns in doing so, since no more than one user can log in to the server at any given time.

    AccMan is one of our core systems, on which a wide variety of issues affecting the IT experience of students, staff and associates can occur. Also it can take a significant amount of time to analyse and resolve some complex account creation issues.

    It would be more efficient for IT Services and would improve the experience of users if multiple user access could be provided to the AccMan Backend Server.
