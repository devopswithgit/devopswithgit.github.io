Accman : Extend Daily

Created by Unknown User (uis894), last modified on Aug 24, 2016

Introduction

Extend Daily maintains the current extensions in Accman. It's a transaction (transaction type 1108) that runs every day at 01:30.

For each extension record it finds (in the EXTENSION table) it checks if it is still valid by checking the extension date. 

If the extension date is in the future, the extension is still valid so the extension is left untouched. However, a quick check on the person status is made and if found to be either Absent or Left, it is updated to Extended.

If the extension date is in the past, the extension has now expired and the person should no longer be at the university. A Cancel Extension transaction (type 40) is created for the affected person. The downstream effect of this is that the account will be disabled.
