JIRA Issue(s) ARF-218 Closed

 

Details

Currently we can only handle 20xx10 term codes however SA are using a number of others, e.g 201507, 201515. Can Accman be updated to handle this. Note the new ESJ project might make this redundant.

To clarify, at any one time, we can only config accman to retrieve students from Banner for a single term code. However for any given academic year, SA could be using 20xx07, 20xx10, 20xx15 etc. The current Banner import process selects students for 20xx10 team.

In addition, students might be setup for future academic years which can cause issues for the current academic year, e.g. students not registered this year but Eligible next year allowing a student to register this year.
