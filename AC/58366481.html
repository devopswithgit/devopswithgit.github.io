Accman : Partner College / Campus Code

Created by Unknown User (uis894), last modified by Unknown User (aaa64656) on Oct 16, 2018

Introduction

The Partner College project introduced a requirement for Accman to capture the campus code for each student. The campus code is stored in Banner and is now copied into Accman during the Banner Import process. 

The development work for this was tracked by JIRA issue  ARF-540 - Campus Code Required by Partner College Closed  where you can see the code changes and some detailed documentation, some of which is duplicated below.

Details

The campus code is stored in Banner's SGBSTDN.SGBSTDN_CAMP_CODE column. Open Athens needs to be able to read the campus code when a student logs in (to Open Athens) so that it can determine what resources the student has access to. Open Athens has the ability to query the Accman database quite easily, so the decision was made to copy the code from Banner into Accman's PERSON table during a Banner Import. To get it there, it is first copied into Accman's BANNER_IMPORT table, because this is where all relevant Banner data is initially copied before it is processed during the Banner Import.
