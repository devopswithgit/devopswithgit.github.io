Accman Refresh : AccMan Script Implementation Effort Matrix

Created by Jon Besson, last modified by Unknown User (ujs077) on Dec 24, 2014

This matrix illustrates the effort required to move each AccMan script into production, and how difficult it might be.

Column Descriptions

Script - script name and JIRA task associated with it's conversion from Perl to C#.

Complexity/Size - see ARF01 - Complexity Matrix. This takes into account dependencies on libraries, database touch points, size of script etc.

Impact/Risk - a description of the riskiness of implementing the script. This includes what effects the script implementation might have on existing production systems.

Testability - A description of how much effort it will take to test the implemented script, whether test beds for connecting systems are available etc.

Roles Affected - who will be affected by the implemented script - Student, Staff, Associates
