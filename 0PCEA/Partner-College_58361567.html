How to use this document

When the release begins, each person named in the Release Steps section should complete their actions in the order and time specified.

If their step is successful, put a tick in the final column and move on to the next step.

If the next step is for someone else to do, add a comment to the bottom of the page, mentioning their name (@richard...) e.g. "Unknown User (uis894), step 2 is now complete, please continue with step 3"

That person will immediately receive an email to inform them that it is their turn.

If a problem is encountered, document it in the final column e.g. "Couldn't connect to the database, this release can't continue" and inform the Key Contact if the issue is serious enough to begin considering a rollback.
